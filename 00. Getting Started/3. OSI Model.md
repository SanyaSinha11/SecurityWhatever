# Overview

**OSI Model** - *Open System Interconnection* 
The Open Systems Interconnection (OSI) model is a conceptual framework that standardizes how different computer systems communicate over a network. It breaks network communication down into 7 distinct layers, ranging from physical hardware (Layer 1) to software applications (Layer 7).

# The 7 Layers of OSI Model
### 7. Application Layer
The topmost layer interacts directly with software applications (like web browsers or email clients) to provide network services

- **Key Functions:** Network process to application, user authentication.
- **Protocols & Entities:** HTTP, FTP, SMTP, DNS.

### 6. Presentation Layer
Ensures data from the sending application is in a readable, usable format for the receiving application. 

- **Key Functions:** Data translation, encryption/decryption, and compression.
- **Protocols & Entities:** SSL, TLS, JPEG, MP3.

### 5. Session Layer
Establishes, manages, and terminates communication sessions between applications. [](https://www.linkedin.com/posts/rocky-bhatia-a4801010_the-7-layers-of-the-osi-model-every-activity-7440359690754138112-V4w5)

- **Key Functions:** Dialogue control (half-duplex/full-duplex) and synchronization (checkpointing).
- **Protocols & Entities:** NetBIOS, RPC.

### 4. Transport Layer
Acts as a liaison between systems, ensuring end-to-end data delivery and data integrity. 

- **Key Functions:** Segmentation of data, flow control, and error recovery.
- **Protocols & Entities:** TCP, UDP.

### 3. Network Layer
Handles logical addressing and determines the best physical path for data (routing) to travel between different networks. 

- **Key Functions:** Routing, packet forwarding.
- **Protocols & Entities:** Routers, IP addresses.

### 2. Data Link Layer
Provides reliable, node-to-node data transfer across the local physical network medium. 

- **Key Functions:** Framing packets, physical (MAC) addressing, and error detection/correction.
- **Protocols & Entities:** Switches, Bridges, MAC addresses, Ethernet.

### 1. Physical Layer
The lowest layer serves as the foundation, converting logical data units into electrical, optical, or radio signals. 

- **Key Functions:** Transmission of raw bit streams over physical media.
- **Protocols & Entities:** Hubs, cables, connectors, network interface cards (NICs).
